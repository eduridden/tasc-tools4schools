# **App Name**: EdTech AI Toolkit

## Core Features:

- AI Tool Listing: Display a searchable and filterable directory of AI tools for educators, including details like app name, description, URL, logo, use case, cost, age restrictions and supporting media.
- Advanced Filtering and Search: Allow users to filter AI tools by use case, subject area, and target audience, and to search using free-form text.
- Logo Fetching: Automatically fetch and display logos for each AI tool based on its URL, using the URL as a tool when reasoning whether it seems like the correct logo to display.
- Detailed Tool Pages: Create individual pages for each AI tool, showcasing all available information, including screenshots and links to videos and documentation.
- Database Storage: Use a Firestore database to store the app details

## Style Guidelines:

- Primary color: Deep sky blue (#3498db) for a professional and trustworthy feel, representing knowledge and innovation.
- Background color: Light gray (#ecf0f1) for a clean and uncluttered interface that focuses on the tools.
- Accent color: Sky blue (#85c1e9) to highlight key elements and calls to action.
- Body and headline font: 'PT Sans', a modern and clear sans-serif, used for both headlines and body text to ensure readability.
- Use flat, simple icons that clearly represent each use case, subject area, and target audience.
- Implement a clean, grid-based layout with ample whitespace to avoid a cluttered appearance and to improve readability.
- Subtle transitions and animations to enhance user experience when filtering, searching, and navigating tool details.
